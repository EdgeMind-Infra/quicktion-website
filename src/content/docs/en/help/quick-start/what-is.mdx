### Q: What is Quicktion?

**A:** Quicktion is an AI Agent marketing platform built for local-business owners. Unlike a generic chatbot, each Quicktion Agent has its own "brain" — it remembers, learns, and operates your computer and browser like a real digital employee.

### Q: What problems does it solve?

**A:** Store diagnostics, creator outreach, competitor analysis, content publishing, ad-campaign planning — every piece of day-to-day marketing work gets done inside a single conversation. A single-location shop can run as efficiently as a franchise.

### Q: Which AI models are supported?

**A:** Several mainstream LLMs are supported. You can see the current model in the bottom-right of the chat window and switch freely in Settings.

### Q: What built-in abilities do Agents have?

**A:** File search, file operations, running commands, scheduled tasks, web lookup, AI image processing, browser automation, user interaction, multi-agent collaboration, and more. Describe what you need in plain language; the Agent picks the right tool.

### Q: Is there an enterprise / team edition?

**A:** An enterprise plan is in design. It's not open yet — stay tuned.

### Q: Is my data safe? Where is it stored?

**A:** Quicktion is local-first: each Agent's core state (memory, config, files) lives on your device. Privacy settings are in "Permissions & Security."
