### 先分析再执行

### 01 店铺诊断 Agent，你的"数据医生"

店铺诊断 Agent 通过深度接入店铺的一手经营数据，可定期给店铺做体检。

全方位扫描店铺的成交表现、流量结构、用户评价等，输出一份"店铺体检报告"——

它能给出可直接行动的诊断结论，让你对症下药。

### 02 经营分分析 Agent，你的"扣分项侦探"

在抖音来客上，有一套"经营分体系"，分数的高低直接影响门店流量。

经营分分析 Agent 通过分析那些模块得分低、怎样提分？然后交给执行 Agent 去处理。

- 门店信息不全？**搭门店 Agent** 来填。
- 达人视频不够？**达人匹配 Agent** 来找。
- 本地推没投？**本地推 Agent** 来投。

它们会根据实际情况行动，而你只需要确认就行。

### 提升抖音经营分，它们是你的手脚

### 03 达人匹配 Agent，找达人，从此有谱了

找达人是很多商家最头疼的事。要么价格难谈；要么花了钱，视频发出来不见效果。

问题不在于达人不好，而是没想清楚：我这次合作的目标、合适的人选。

达人匹配 Agent 会给出合适的达人营销方案，然后帮你找达人、谈细节、跟进效果。——全链路闭环，你不用守在电脑前一个个谈。

### 04 竞品分析 Agent，你的"市场情报员"

想知道同城店铺最近发了什么爆款视频吗？参与的促销活动？近期哪种品类大受欢迎？

在以前这些要靠你去刷，十分费时间。

现在竞品分析 Agent 会帮你监控竞品营销动态：爆款内容、市场热点、竞品情报，从此你的运营内容跟着市场走，不再闭门造车。
