### 01 在手机上能干什么？

连接之后，你在微信/飞书里能做的，和在电脑上几乎一样，功能不打折：

- **下达任务**：告诉它去执行任务
- **查看进度**：随时提问当前任务进度
- **接收成果**：方案文档、分析报告直接发给你
- **修改调整**：对结果不满意，可在对话里说明，AI 重新生成
- **确认执行**：需要你拍板的步骤，它会在微信里等你确认

一切都在你习惯的聊天界面里完成，不需要切换任何 App。

但如果你想回溯整个任务过程，可登录平台查看，细节一样不会丢失。这里 Agent 干活的每一步都实时展示给你看。

### 02 适合什么样的商家？

- **门店经营忙**：根本没时间坐在电脑前，但手机不离手
- **多店管理**：几家分店同时跑，地点常换，手机操控很方便
- **喜欢即时响应**：任务完成立刻知道，不用登录平台看进度
- **团队协作**：老板和运营都在群里，接收的成果大家一起看

### 03 账号安全有保障吗？

你放心，我们不需要你提供账号密码，不会存储通讯信息。

AI 只是接收指令、发送成果给你——它不能查看你的聊天记录，不能代替你发消息给他人。

你可随时断开连接，取消授权。

:::tip
AI 最大的价值，是让你可随时启动，随时收结果，而不是绑定你坐在电脑前。
:::
