import { withBase } from "../../../../lib/paths";

很多老板问：让 AI 帮我管抖音，第一步该干啥？

很简单：把抖音来客账号授权给巨象平台，即可接管抖音运营。

### 01 找到入口，点击进入

**1** 登录巨象平台网页端，在左侧菜单找到「来客店铺授权」按钮，点击进入店铺授权页。

<img src={withBase("/images/help/douyin-auth/0-entry.png")} alt="找到授权入口" />

**2** 在会话窗口里发送授权请求。

<img src={withBase("/images/help/douyin-auth/1-request.png")} alt="点击来客授权" />

### 02 登录来客账号

**3** 右侧浏览器已打开抖音来客登录页面。在交互卡片中输入来客绑定的手机号，点击提交。→ 当然你可以在右侧浏览器中操作，输入手机号和验证码。

<img src={withBase("/images/help/douyin-auth/2-phone.png")} alt="输入手机号" />

**4** 发送短信验证码后，填写验证码并点击提交。

<img src={withBase("/images/help/douyin-auth/3-otp.png")} alt="输入验证码" />

### 03 选择默认运营门店

**5** 成功登录来客后，Agent 会自动读取你账号下的所有店铺，你需要选择「默认运营门店」——后续所有诊断、达人匹配、商品上架等操作，Agent 都会优先针对这家店执行。当然，随时可以切换。

<img src={withBase("/images/help/douyin-auth/4-pick-store.png")} alt="选择门店" />

**6** 店铺信息已经写入巨象的记忆系统，你的 AI 运营官正式上岗。接下来，你只需要在微信里发句话：「帮我诊断店铺」，它就能自动干活了。

<img src={withBase("/images/help/douyin-auth/5-memory.png")} alt="写入记忆" />

:::tip
授权一次，长期有效。可添加更多账号，随时在会话里授权即可。
:::
