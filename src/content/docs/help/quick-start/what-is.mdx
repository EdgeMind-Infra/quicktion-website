## Q：Quicktion 是什么？

**A：** Quicktion 是一个面向本地生活商家的 AI Agent 营销平台，不同于普通聊天机器人，它的 Agent 拥有独立的「大脑」，能记忆、能学习、能操作你的电脑和浏览器，像一个真正的数字员工。

## Q：它主要解决什么问题？

**A：** 店铺诊断、达人对接、竞品分析、内容发布、投流策划——所有日常营销工作都在对话中完成，让单店运营也能像连锁品牌一样高效。

## Q：支持哪些 AI 模型？

**A：** 支持多种主流大模型，可在对话框右下角查看，用户可在设置中自由切换。

## Q：Agent 有哪些内置能力？

**A：** Agent 常见能力包括：文件搜索、文件操作、运行命令、定时任务、网络检索、AI 图像处理、浏览器自动化、用户交互、多 Agent 协作等。用户只需用自然语言描述需求，Agent 会自动选择合适的能力。

## Q：有企业版/团队版吗？

**A：** 关于企业级套餐，目前我们正处于方案设计与功能打磨阶段，暂未开放。

## Q：数据安全吗？存在哪里？

**A：** Quicktion 采用本地优先架构，Agent 的核心数据（记忆、配置、文件）存储在你的本地设备上。你可以在「权限与安全」中管理数据隐私设置。
